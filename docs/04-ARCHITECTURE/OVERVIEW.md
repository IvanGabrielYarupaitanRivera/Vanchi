## Modos de renderizado

| Ruta | Modo | Detalle |
|------|------|---------|
| `/` (Home) | SSR | Renderizado en servidor, datos estГЎticos |
| `/proyectos` | SSR | Listado desde data layer |
| `/proyectos/[projectId]` | SSR | Carga dinГЎmica vГӯa `+page.ts` + `load()` |
| `/precios` | SSR | Datos estГЎticos |
| `/soluciones-legales` | SSR | Datos estГЎticos |
| `/waas` | SSR | Datos estГЎticos |
| `/sitemap.xml` | Prerendered | Generado en build, sirve XML estГЎtico |
| PГЎginas legales | SSR | Contenido estГЎtico |

## Estructura en capas

```
Capa 1 вҖ” Routes (SvelteKit)
в”ңв”Җв”Җ +layout.svelte      вҶ’ Layout global (Header + Footer + children)
в”ңв”Җв”Җ +error.svelte       вҶ’ PГЎgina de error 404/500
в””в”Җв”Җ [ruta]/+page.svelte вҶ’ PГЎgina especГӯfica

Capa 2 вҖ” Components (Svelte 5)
в”ңв”Җв”Җ lib/components/     вҶ’ Componentes reutilizables
в”Ӯ   в”ңв”Җв”Җ Header.svelte   вҶ’ NavegaciГіn
в”Ӯ   в”ңв”Җв”Җ Footer.svelte   вҶ’ Footer
в”Ӯ   в”ңв”Җв”Җ SEO.svelte      вҶ’ Meta tags
в”Ӯ   в””в”Җв”Җ [seccion]/      вҶ’ Componentes agrupados por secciГіn

Capa 3 вҖ” Data Layer (TypeScript)
в”ңв”Җв”Җ lib/data/projects/  вҶ’ Datos de proyectos (estГЎticos)
в”ңв”Җв”Җ lib/services/       вҶ’ Servicios que consultan los datos
в”ңв”Җв”Җ lib/types/          вҶ’ Interfaces y tipos
в””в”Җв”Җ lib/constants/      вҶ’ Constantes del proyecto

Capa 4 вҖ” Assets
в”ңв”Җв”Җ lib/assets/images/  вҶ’ ImГЎgenes WebP
в”ңв”Җв”Җ lib/assets/icons/   вҶ’ SVGs (network, technologies, logos)
в””в”Җв”Җ static/             вҶ’ Favicon, archivos estГЎticos
```

## ConfiguraciГіn del build

```
Vite (vite.config.ts)
в”ңв”Җв”Җ @tailwindcss/vite          вҶ’ Procesa Tailwind v4
в”ңв”Җв”Җ @sveltejs/kit/vite         вҶ’ Plugin SvelteKit
в””в”Җв”Җ @sveltejs/enhanced-img     вҶ’ OptimizaciГіn de imГЎgenes

SvelteKit (svelte.config.js)
в”ңв”Җв”Җ adapter: @sveltejs/adapter-vercel  вҶ’ Despliegue en Vercel
в””в”Җв”Җ preprocess: vitePreprocess         вҶ’ Procesa TS/SCSS
```
