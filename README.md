# Vanchi

> Portafolio profesional de **Ivan Gabriel Yarupaitan Rivera** — desarrollador full-stack y consultor tecnologico.

## Stack

| Capa      | Tecnologia                   |
| --------- | ---------------------------- |
| Framework | SvelteKit 2 (Svelte 5)       |
| UI        | Svelte 5 (runes)             |
| Estilos   | Tailwind CSS v4 + DaisyUI v5 |
| Backend   | Convex Cloud                 |
| Runtime   | Bun                          |
| Hosting   | Vercel + Convex Cloud        |

## Comandos rapidos

```bash
bun dev          # Servidor de desarrollo
bun build        # Build de produccion
bun run check    # Type-check + lint
bun run format   # Prettier --write
```
