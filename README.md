## Stack

| Capa      | Tecnología                   |
| --------- | ---------------------------- |
| Framework | SvelteKit 2                  |
| UI        | Svelte 5 (runes)             |
| Estilos   | Tailwind CSS v4 + DaisyUI v5 |
| Runtime   | Bun                          |
| Hosting   | Vercel                       |

## Comandos rápidos

```bash
bun dev          # Servidor de desarrollo
bun build        # Build de producción
bun run check    # Type-check + lint
bun run format   # Prettier --write
```
